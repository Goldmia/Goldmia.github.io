有哪些好看的 Hexo 主题？

当然这些主题我没有一个一个的试，我只是选了排行榜上比较靠前，然后看了看效果，觉得不错，我就下载下来了。

这里，我就讲一下我所使用的yilia主题吧。

yilia主题下载地址

安装使用

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

注意：这里最好把它下载到博客所在地址，hexo目录下面的themes文件夹里面。

配置

打开博客所在位置，修改hexo根目录下得 _config.yml : theme: yilia

更新

cd themes/yilia git pull

这样主题就使用成功了，打开你的博客，你就看到博客主题换成了yilia的主题。然后下面再配置一下你的个人信息，比如你的微博，git地址什么的。当然还有人会问怎么换头像啊，下面就讲一下这些问题。
